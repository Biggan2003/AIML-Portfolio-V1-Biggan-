## 🎯 About The Project

This is my personal portfolio website that represents my professional identity as an **AI/ML Engineer**. It highlights my technical skills, professional experience, featured projects, certifications, and community contributions.

The portfolio features a **dynamic animated background** with glowing particles, **dual orbital tech logos**, and a fully responsive design that works seamlessly on all devices.

### 🎨 Design Highlights

- ✨ **Vibrant animated particle network** with connecting lines
- 🌈 **Cyan + Purple gradient glow effects**
- 🪐 **Dual orbital rings** (AI/ML tools + Non-AI/ML tools)
- 🎭 **Dynamic icon cycling** in About section cards
- 📱 **Fully responsive** (Mobile, Tablet, Desktop)
